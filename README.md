Retail-Sales-Performance-Dashboard
This Power BI project analyzes retail sales data to provide insights into:
- Sales performance
- Customer trends
- Store performance
- Payment method analysis
- Peak sales periods
  
Key Features
- Interactive slicers
- KPI cards
- Trend analysis
- Store comparison
- Time-based sales insights

Tools Used
- Power BI
- Excel / CSV dataset

How to Use
1. Download the `.pbix` file.
2. Open it using Power BI Desktop.
3. Interact with filters and visuals.
