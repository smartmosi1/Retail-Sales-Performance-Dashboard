RETAIL SALES PERFORMANCE DASHBOARD (POWER BI)
This project presents an interactive Power BI dashboard that analyzes retail sales transactions to generate actionable business insights across sales performance, customer behavior, store operations, and payment methods.

PROJECT OVERVIEW
The dashboard answers key business questions such as:
- Which stores are performing best?
- What are the peak sales periods?
- How do customer trends affect revenue?
- Which payment methods are most used?
- Are sales declines linked to customer loss?
It enables managers and decision-makers to optimize staffing, inventory, marketing, and customer retention strategies using data-driven insights.

KEY FEATURES
- KPI cards for total sales, average sales, quantity sold, and customers
- Sales trend analysis by month and hour
- Store performance comparison
- Payment method distribution
- Customer behavior and retention analysis
- Interactive slicers for filtering by time and category

TOOLS USED
- Power BI Desktop
- Microsoft Excel / CSV Dataset

REPOSITORY CONTENTS
- Retail_Sales_Dashboard.pbix — Power BI report file   
- Screenshots — Dashboard images  
- README.md — Project documentation  

HOW TO USE
1. Download the .pbix file.
2. Open it using Power BI Desktop.
3. Interact with slicers and visuals to explore insights.
